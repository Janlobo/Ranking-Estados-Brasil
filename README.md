# Ranking_States_Brazil
Classification of Brazilian states based on 12 key indicators.

# What the project does  
<p>A unified view of the situation of Brazilian states based on 12 pre-determined indicators was requested.</p><br>  
<p>The dashboard should have a single page and display the most recent data for each indicator, the trend of the numbers, and their overall ranking in relation to other states.</p>  
<p>In addition to classification by indicators, it was requested to create a ranking of the states in two ways:<br><br>  
    . Classification of the best states with scores from 0 to 5, where 5 is the best.<br>  
    . Classification of states with the best growth trends, scored from 0 to 100.</p>  

# Why the project is useful  
<p>The dashboard was created by accessing data directly from official sources such as IPEA, IBGE, DATASUS, Ministry of Justice, etc.<br>  
    It provides an overview of the state and how the governor's management is performing during this period.</p>  

# How users can start using the project  
<p>The dashboard was created using Power BI. To visualize it, you need to access the panel through the Power BI service or open it locally using Power BI Desktop.</p>  
<p>Before downloading the .PBIX file, please request permission from Jan Lobo, informing that you intend to download the file.</p>  

# Where users can get help with this project  
The project was created and developed by Jan Lobo.  

# Skills and resources used in the project  
<li>Access to databases through <b>APIs</b> and Excel files made available by entities on their digital platforms.</li>  
<li>Use of <b>Parameters</b> to select the most recent year.</li>  
<li>Data processing (ETL) using step-based transformations and programming in <b>Query M</b>.</li>  
<li>Creation of 81 measures with <b>DAX</b> for classifications and chart generation.</li>  
<li>Data model using 24 tables in a star schema.</li>
